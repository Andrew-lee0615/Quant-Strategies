# 📈 BTC-USD Strategy: MDD 30% Challenge & Momentum Expansion

이 프로젝트는 비트코인(BTC)의 극심한 변동성을 제어하면서도 대세 상승장의 수익을 온전히 누리기 위한 **추세 추종(Trend-Following)** 및 **리스크 관리(Risk-Off)** 전략을 다룹니다.

## 1. 백테스트 결과 요약 (Backtest Performance)

두 전략 모두 동일한 결과가 도출되었음. 이는 역사적으로 비트코인이 초과열 구간(이격도 1.6 이상)에서 깊은 눌림목(MA20/60 터치)을 주지 않고 수직 상승하거나, 조정 시 이격도를 빠르게 해소하며 하락했음을 의미함.

| Metric | Version 1 (1.6 Limit) | Version 2 (2.0 Expansion) |
| :--- | :--- | :--- |
| **Total Return** | **10600.03%** | **10600.03%** |
| **CAGR** | **59.57%** | **59.57%** |
| **Max Drawdown (MDD)** | **-36.91%** | **-36.91%** |

## 2. 전략 비교 (Strategy Comparison)

### 공통 기본 로직
- **진입:** MA120 돌파 후 5일 유지 또는 +5% 돌파 시 (현금 20%)
- **수익 실현:** MA10 대비 +8%(15% 매도), +15%(30% 매도) 가속 익절
- **리스크 오프:** MA10 < MA20 데드크로스 시 20% 선제 매도, MA120 이탈 시 단계적/전량 탈출

### 이격도 필터 차이점
#### [V1] 보수적 이격도 제한 (Disparity Limit 1.6)
- **핵심:** "너무 비싼 구간에서는 절대 추가 매수하지 않는다."
- **매수 제한:** 주가와 MA120의 이격도가 1.6(60% 차이)을 초과하면 모든 눌림목 매수를 중단함.
- **장점:** 상투를 잡을 확률을 원천 차단하여 MDD를 극도로 방어함.

#### [V2] 확장형 모멘텀 (Momentum Expansion 2.0)
- **핵심:** "모멘텀이 강하면 비싸더라도 비중을 줄여서 참여한다."
- **매수 제한:** 이격도 제한을 **2.0(100% 차이)**까지 확장.
- **조심 매수 구간:** 이격도 1.6~2.0 사이에서는 눌림목 매수 비중을 **절반(50%)으로 축소**하여 진입.
- **장점:** 초과열 불장에서의 소외(FOMO)를 방지하고 추가 수익 기회를 탐색함.

## 3. 세부 매매 로직 (Trading Rules Table)

| 구분 | 조건 (Conditions) | 행동 및 비중 |
| :--- | :--- | :--- |
| **신규 진입** | MA120 상향 돌파 (Confirm: 5일 유지 or +5% 돌파) | 현금의 **20%** |
| **안전 매수** | 이격도 ≤ 1.6 구간에서 MA20/MA60 터치 (양봉 확인) | MA20(**5%**) / MA60(**10%**) |
| **조심 매수** | 1.6 < 이격도 ≤ 2.0 구간에서 MA20/MA60 터치 (양봉 확인) | **비중 50% 축소** (2.5% / 5%) |
| **선제 타격** | **MA10 < MA20** (단기 추세 꺾임) | 보유량의 **20% 매도** |
| **메인 탈출** | 주가 < MA120 (장기 추세 붕괴) | **50% 매도 → 95% 지점 전량 매도** |

## 4. 결론 및 인사이트
1. **MDD 방어력:** 선제 타격(MA10 < MA20) 로직이 비트코인의 대폭락 직전에 비중을 성공적으로 줄여 MDD를 **-30%대**로 유지하는 데 핵심적인 역할을 함.
2. **이격도 유효성:** 비트코인 역사에서 이격도 1.6 이상의 구간은 매우 짧고 강력한 '피날레' 구간임. 이 구간에서 조정을 기다리기보다 기존 물량을 익절하는 것이 결과적으로 동일한 퍼포먼스를 내면서 리스크를 줄이는 길임을 데이터로 확인.
